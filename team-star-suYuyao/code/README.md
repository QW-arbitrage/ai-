## 📈 第二周相对于第一周的改进

| 改进项 | 第一周 | 第二周 |
|--------|--------|--------|
| **候选文本补齐** | 仅提取部分段落，遗漏关键增资事件 | 补齐所有候选文本段落，确保事件覆盖完整 |
| **evidence_text字段** | 未包含原文依据，无法溯源 | 每个事件/股东条目均附带evidence_text，标注招股说明书原文 |
| **source_page字段** | 未标注页码 | 每条记录标注source_page，可回溯至招股说明书对应页 |
| **Pydantic数据校验** | 无结构化校验，JSON格式可能不规范 | 引入Pydantic模型进行字段类型、必填项、逻辑一致性校验 |
| **cross-check日志** | 无交叉校验 | 增加cross-check日志，对比流量表与存量表数据一致性 |

## 📝 独立完成说明

本仓库所有代码及数据均由苏宇垚个人独立完成：

1. **代码框架**参考了课程提供的模板结构，但数据处理逻辑、校验规则、输出格式均为本人独立编写
2. **8家公司**的招股说明书下载、PDF解析、关键段落定位、字段提取、Pydantic校验全部由本人独立执行
3. **所有输出数据**（认缴流量表、股权结构存量表）均为本人独立提取和校验的结果
4. **cross-check日志**中发现的差异也由本人独立排查和修正

如需核实，可对照各公司的招股说明书原文及source_page字段进行查证。
